UNREAL ENGINE · REALTIME VIZ

Architecture Visualisation

A real-time architectural walkthrough built and rendered live in Unreal Engine — demonstrating spatial layout, lighting, and materials in an interactive environment.

Role

Designer & developer

Context

University of Portsmouth

Engine

Unreal Engine

Type

Real-time visualisation

Overview

This project explores how Unreal Engine's real-time rendering pipeline can be used to produce architectural visualisations that go beyond static renders — letting viewers navigate the space interactively and experience lighting, scale, and materials as they would in a game engine.

The work sits at the intersection of spatial design and game-engine craft, using the same tools and thinking I bring to any immersive experience: lay out the space, light it, and iterate until it reads correctly in motion.

What I built

  • Modelled and dressed the environment using Unreal Engine's toolset, focusing on spatial legibility and material fidelity.
  • Set up a real-time lighting rig — natural and artificial — to give the space mood and depth rather than a flat, static look.
  • Packaged the result as a navigable walkthrough, not just a flythrough render.
